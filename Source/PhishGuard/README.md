# PhishGuard 🎣

PhishGuard is the phishing URL detection and suspicious web link analysis module of ForensicBox. It helps identify commonly used phishing indicators inside suspicious URLs and domains.

---

## 🔍 Module Purpose

Phishing remains one of the most common cyberattack vectors.

This module allows analysts and students to inspect links for:

- fake login attempts
- malicious redirections
- suspicious domain tricks
- credential harvesting indicators

---

## ⚙️ Features

- Suspicious URL Detection
- Phishing Keyword Inspection
- IP Address URL Checks
- Long Abnormal Link Analysis
- Fake Login Pattern Recognition
