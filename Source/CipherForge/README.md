# 🔐 CipherForge:

CipherForge is the cryptographic utility module of the ForensicBox cybersecurity toolkit. It is designed to demonstrate practical encryption and decryption workflows commonly used in secure communications and cybersecurity investigations.

---

## 🔍 Module Purpose

Cybersecurity analysts often encounter encoded, encrypted, or obfuscated text during:

- forensic investigations
- malware analysis
- credential reviews
- secure data validation

CipherForge provides Python-based utilities to simulate and understand these cryptographic processes.

---

## ⚙️ Features

- Text Encryption
- Text Decryption
- Basic Cipher Demonstrations
- Encoded Data Handling
- Secure String Transformation
