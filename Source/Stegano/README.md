# Stegano 🕵️

Stegano is the hidden data and steganography forensic analysis module of the ForensicBox toolkit.

It is designed to inspect files for concealed or embedded suspicious content.

---

## 🔍 Module Purpose

Attackers sometimes hide data inside normal-looking files such as:

- images
- text files
- encoded payloads
- metadata containers

Stegano helps demonstrate the first-level forensic process of hidden artifact discovery.

---

## ⚙️ Features

- Hidden Content Investigation
- Suspicious Embedded Data Detection
- File Artifact Inspection
- Basic Steganography Awareness
